I first wrote this game in C# and I am working on deploying it in web, and mobile app. The game is simple, all you need to do is figure out the different codes (0~9) and their position generated by game system. When the code is correct and in its position, then the system will give you (1A). If the code is correct but not in the right position, then the system will give you (1B). Game system will tell you how many(1A) or (1B) you got, right after your every guess.

Guess the three digit code to defeat the computer!
